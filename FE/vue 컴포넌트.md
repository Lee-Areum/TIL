## VueJS에서 컴포넌트 만들어서 사용하기

vue에서 컴포넌트를 사용하려면

1. 컴포넌트로 사용할 파일 생성
2. 해당 컴포넌트를 사용할 파일에 import 해주고 components에 추가


### 데이터 이동 방법
1. 메인화면 -> 컴포넌트
- props를 사용하여 데이터 이동
메인 화면에서는 태그의 요소로 props 넘김
컴포넌트에서는 props로 데이터 받음
```
// 메인
<component
  prop1 = "prop1"
/>

// 컴포넌트
props(){
  prop1: {
    type: String
  }
}
```

2. 컴포넌트 -> 메인화면
- emit을 사용하여 데이터 이동
메인 화면에서는 @를 사용하여 이벤트를 받음
컴포넌트에서는 emit("이벤트 이름", 데이터)를 사용함
